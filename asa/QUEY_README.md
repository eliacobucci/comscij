## What is Quey?

Quey is a professional tool for creating paired comparison questionnaires used in multidimensional scaling research. It generates beautifully formatted questionnaires with all possible concept pairs, following established research methodology.

### Key Features
- ✅ **Professional formatting** with Times New Roman font
- ✅ **Blocked layout** (7 questions per block with reference reminders)
- ✅ **Answer boxes** instead of underlines
- ✅ **Multiple export options** (Text, CSV, Print, Copy)
- ✅ **No installation** - single HTML file
- ✅ **Works offline** - no internet required

## How to Use Quey

### Step 1: Enter Your Concepts
**Choose your method:**
- **Type concepts** individually (with add/remove buttons)
- **Paste a list** (one concept per line)

### Step 2: Set Reference Pair
- Choose two concepts from your list
- These become the "100 units apart" examples in instructions
- Pick concepts that represent meaningful extremes

### Step 3: Generate & Export
- Click "Generate Questionnaire"
- Choose your export format:
  - **Text file** - Ready for printing
  - **CSV file** - For online surveys
  - **Print** - Direct professional printing
  - **Copy** - Paste into other applications
